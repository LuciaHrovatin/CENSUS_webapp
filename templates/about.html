C.E.N.S.U.S

Calculator of Earnings Nationally Scored via User Specifics


CENSUS is a web application, developed using Flask framework, that aims at informing the user about his/her income bracket, predicted on the basis of some of his/her census data.
We are Lucia and Elisa, two Data Science students at University of Trento. This project has been developed during Big Data Technologies course and the aim is to build a Big Data System able to predict the IRPEF income groups based on some census data:

  • age (more precisely year of birth)
  • gender (optional, the user can not specify it)
  • province of residence (ISTAT code)
  • number of family component
  • marital status (single, cohabitee, married, widow, separated, divorced)

The prediction is made possible by a Random Forest algorithm fed with the entered data. Such algorithm has been trained on some big Banca d'Italia and Sole 24 Ore datasets:

  • Banca d'Italia, Indagine sui bilanci delle famiglie italiane, Indagine sul 2018
  • Banca d'Italia, Indagine sui bilanci delle famiglie italiane, Indagine sul 2016
